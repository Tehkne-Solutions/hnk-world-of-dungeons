# SPRINTS

## Sprint 0 — Compilação e saneamento técnico

Objetivo:
Fazer o projeto compilar no Dream Maker e gerar o .dmb.

Tarefas:

- corrigir assets faltantes;
- validar includes;
- remover duplicidade de world/New();
- validar paths;
- validar macros;
- compilar sem erros.

Critério de pronto:

- hnk_world_of_dungeons.dmb gerado;
- 0 erros de compile;
- warnings documentados, se existirem.

## Sprint 1 — Protótipo jogável v0.1

Objetivo:
Permitir jogar o loop Auralis → missão → Fenda → combate → coleta → retorno → entrega.

Tarefas:

- testar spawn;
- testar NPC Oficial de Contratos;
- testar missão Rank D;
- testar portal;
- testar mobs;
- testar coleta;
- testar recompensa;
- testar memória narrativa simples.

Critério de pronto:

- loop completo jogável sem intervenção manual.

## Sprint 2 — UX de teste e feedback

Objetivo:
Melhorar clareza para testadores.

Tarefas:

- mensagens de orientação;
- verb Status melhorado;
- placas ou NPCs explicativos simples;
- feedback de missão ativa/concluída;
- documentação de QA.

Critério de pronto:

- jogador entende o fluxo sem orientação externa.

## Sprint 3 — Base de progressão

Objetivo:
Começar progressão real de personagem, skill e guardião.

Tarefas:

- XP de personagem;
- XP de skill;
- vínculo do guardião;
- Sincronia melhorada;
- custo/cooldown mais claro;
- estrutura inicial de save.

Critério de pronto:

- ações do jogador geram progresso persistente ou preparado para persistência.
