# HNK: World of Dungeons

Protótipo inicial em BYOND para o projeto **HNK: World of Dungeons**.

## Objetivo da primeira fase

Implementar uma base mínima e expansível para validar o loop:

`Auralis -> contrato Rank D -> Portão da Fenda -> Entrada Rachada -> combate -> coleta -> retorno -> recompensa -> memória`

## Primeiros sistemas

- Estrutura modular do projeto BYOND.
- Player com recursos básicos: HP, Energia Física, Fluxo Espiritual e Sincronia.
- Combate simples com dano.
- Skill inicial: `skill_fogo_faisca_ardente`.
- Guardião inicial: `guardiao_salamandra_animica`.
- Mob inicial: `mob_primeira_fenda_rastejante_pedra`.
- Missão Rank D: `missao_d_primeira_fenda_limpeza_entrada`.
- Evento narrativo simples: `evento_missao_rank_d_concluida`.

## Convenção

IDs técnicos usam letras minúsculas, sem acentos e com underline.
